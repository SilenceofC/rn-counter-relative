## 安装及运行：
##### 1. 下载
```bash
$ git clone git@github.com:uniquexiaobai/rn-counter-relative.git
```
##### 2. 安装依赖
```bash
$ cd rn-counter-relative && yarn
```
##### 3. 运行
- Android版运行：
```bash
$ react-native run-android
```
- IOS版运行：<br/>
双击 `~/ios/rn-counter-relative.xcodeproj` 文件，默认用 `Xcode` 打开，点击运行按钮即可
